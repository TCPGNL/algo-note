# define 和const 的区别
define是宏定义 而 const 是常变量 宏定义是直接对相应的部分替换 然后才进行编译和运行。
```cpp
#include<iostream>
#define CAL(x) (x*2+1)

int main()
{
    int a=1;
    printf("%d\n",CAL(a+1));
    return 0;
}
```
